2017 – 2018 Reconstruction of transparent objects SIGGRAPH 2018 Project
Transparent objects usually are hardly to be captured by traditional optical devices. One possible direction is to explore the rays involved in reflections and refractions. An algorithm that combines analyzing ray behaviors and image-based modeling is proposed to reconstruct the surface of transparent objects.

2016 – 2016 Plate mode NURBS ray-tracing Google Summer of Code
Supported by BRL-CAD in Google Summer of Code(GSoC)
The boundary representation is a method for representing 3D objects. But for solid modeling system, it is impractical to use these thin surfaces to describe a solid mesh. An available approach is proposed that assigning an implicit thickness to the surface and making it as a pseudo-solid object, we call it plate mode.

2015 – 2016 Functionality analysis of 3D objects SIGGRAPH 2016 Project
By learning a functionality model, which encodes a mapping from object geometric features to specific functionality, for an object category via co-analysis of object-to-object interactions. Then given a single 3D object, we can predict its functionality and locate the functional regions it can support.
